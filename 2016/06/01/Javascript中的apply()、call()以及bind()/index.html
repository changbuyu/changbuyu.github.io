Javascript中的apply()、call()以及bind()

发表于 2016-06-01   |   分类于 Javascript   |  

最近在学习JS，刚刚入门，勉强分清楚了apply()、call()以及bind()，熟话说好记性不如烂笔头。写个笔记，留给自己以后看。

1. 先看看this关键字

this引用的是函数执行的环境对象——或者也可以是this的值。 ——《Javascript高级编程》

讲真，这句话我没太看懂，按照我自己的理解，谁调用的函数，那么谁就是这个this，依旧用书中的例子：

归纳成一句话：当函数被执行时，拥有它的object将被传入作为this。

2. 说说apply()与call()

这两个只需要理解一个即可，因为这俩哥们的用法是一样的，区别在于接收参数的方式不用。他俩的第一个参数是一样的：在其中运行的作用域(下面会有细说)。apply()只接受两个参数，第二个参数接受数组作为参数，而call()只能直接传递给函数，不能传递数组，且对数量没有限制。

当然这并不是apply()与call()的主要用途，这哥俩的强大之处在于扩充函数赖以运行的作用域，按照我自己的理解就是让一个对象可以调用另外一个对象的方法。
